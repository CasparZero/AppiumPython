#### Use Appium ####
1. Install nodejs.
$brew install node

2. Install appium.
$npm install -g appium
Note:
Not need install this with sudo.

3. Launch appium.
$appium & or appium -a 127.0.0.1

####  Use Python ####
1. Install Python.
$brew install python

2. Install appium python client.
$pip install appium-python-client.
